Student Finance Manager

Documentation & Detailed Explanation

📌 Introduction

The Student Finance Manager is a web-based system designed to help students efficiently manage their financial records, expenses, budget planning, and even track their wellness logs. It provides an interactive dashboard where students can monitor their spending habits alongside their health and study activities.

⚙️ Core Modules

  • Login & Authentication – Secure student login with password or social login.
  • Finance Tracking – Add, edit, and monitor daily/weekly/monthly expenses.
  • Budget Planner – Define budgets and track progress.
  • Wellness Monitor – Log sleep, stress, study hours, and exercise activities.
  • Reports & Graphs – Generate interactive charts for both finance and wellness data.

💻 Technologies Used

  • HTML5 & CSS3 – For structure and styling.
  • JavaScript – For interactivity and dynamic updates.
  • Chart.js – To display finance and wellness data visually.
  • Flask / Node.js (optional backend) – To store data and manage authentication.

🚀 Features in Action

  • 📊 Interactive Graphs for expense trends and wellness logs.
  • 🔑 Secure Login with session/token support.
  • 💸 Expense Categories like food, books, travel, and fees.
  • 🧘 Wellness Insights (sleep vs stress balance).
  • 📥 Export Options (PDF/Excel/PNG reports).

📂 Example Workflow

  1. Student logs in via the login.html page.
  2. They enter daily expenses and view balance updates.
  3. They log wellness activities (sleep, exercise, stress).
  4. Dashboard combines finance + wellness for better insights.
  5. Reports can be exported and shared.

🔒 Security Notes

- Passwords are stored securely (hashed on backend).
- Sessions or JWT tokens protect user data.
- Logout clears local storage and ends session.

📈 Example Integration

Example chart in Chart.js for expenses:


    const data = {
      labels: ['Food', 'Books', 'Travel', 'Fees'],
      datasets: [{ label: 'Expenses', data: [1200, 800, 450, 2000] }]
    };
    new Chart(ctx, { type: 'pie', data });

✅ Conclusion

The Student Finance Manager simplifies financial and wellness tracking for students in a single dashboard. It is extendable, secure, and integrates both lifestyle and expense management for holistic student growth.
